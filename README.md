# Task Tracker + Calendar

A sleek task manager with Firebase auth + Firestore database.

## Features
- Email/password login
- Add tasks with due dates & notes
- Mark as done / delete
- Real-time sync using Firestore
- Modern grey-black UI

## Setup
1. Create Firebase project
2. Enable Email/Password auth
3. Enable Firestore
4. Replace firebaseConfig in `app.js`
